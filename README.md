# homebrew-tap

A Homebrew tap for software and tools by me.

## Usage

```
brew tap wassimk/tap
```

## Available Formulae

| Formula | Description |
| --- | --- |
| `granary` | Export Granola meeting notes and transcripts to markdown |
| `tmignore` | Exclude developer dependency directories and arbitrary paths from macOS backups |
| `st` | CLI for managing Git stacked branches |
| `elgato-light` | CLI to control Elgato Key Light and Key Light Air devices |
| `elgato-autolight` | Automatically toggle Elgato lights when your Mac camera activates |

### Install

```
brew install wassimk/tap/granary
brew install wassimk/tap/tmignore
brew install wassimk/tap/st
brew install wassimk/tap/elgato-light
brew install wassimk/tap/elgato-autolight
```
